# Requirements

 - Raspberry Pi Zero with wifi and presoldered headers
 - Adafruit OLED display bonnet with buttons
 - MicroUSB to USB adapter
 - Small USB connected soundcard
 - Raspbian OS
 - NodeJS version 9.5 or later
